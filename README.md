# Nearby Venues

## Requirements

- Node.js

## Installation

- Clone this repo

  `git clone git@github.com:ctram/points-of-interest.git`.

- Change into repo directory

  `cd points-of-interest/`

- Install dependencies,

  `npm install`

- Bundle JavaScript

  `npm run build`.

- Start server

  `npm start`.

- Open browser and navigate to `localhost:3000`
